Project Overview
African Data Foundation web application was developed with the Django sitting as the base framework using markup languages such as HTML, CSS, and  JavaScript for the frontend. This project involves several stages, including planning, design, development, testing, launch, and maintenance. During the planning stage, the project scope, objectives, requirements, and timelines are defined, and a strategy is created to achieve the project goals. Webflow was used in structuring the entire site for time efficiency between the idea and the execution.

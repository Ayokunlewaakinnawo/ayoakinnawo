Project 3

Data Mining / Engineering
Project Overview
This project is a concise demonstration of the comprehensive continuous data mining and engineering project I worked on at Vox Technologies. Throughout the project, my main objective was to develop scripts that would facilitate the creation of new data pipelines to provide valuable insights into inventory management. Additionally, I successfully created data pipelines for order tracking, quote requests, and other relevant aspects of the business operations.One of the crucial tasks was transforming the newly developed data pipelines into visually appealing presentations for executives and stakeholders. To accomplish this, I utilized various tools and technologies. Azure Data Studio played a pivotal role in writing queries and extracting data from the database. Jupyter Notebook provided a robust development environment for data exploration and analysis. For handling large-scale data processing, I utilized Pyspark and Snowflakes. Finally, Power BI served as the go-to tool for data visualization and generating insightful reports. To ensure efficient collaboration and version control, I leveraged Git for sharing code and facilitating collaboration among team members.Overall, this project exemplified my expertise in data mining, engineering, and visualization while utilizing a range of powerful tools to deliver valuable insights to the executive team and stakeholders at Vox Technologies.
My Contributions
As the sole proprietor of this project, I was responsible for overseeing and actively participating in all aspects of its execution. This included preliminary preparation, where I generated ideas and devised strategies for accomplishing each task within the project. I also collaborated closely with the sales team to conduct internal research, aiming to gain a thorough understanding of the high-priority key performance metrics that would drive efficient workflows.
